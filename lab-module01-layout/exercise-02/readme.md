# Ejercicio 2: Crear dos temas distintos y mostrar los resultados en una página.
Los cambios en el tema afectan a las siguientes características:
- Color
- Fuente
- Border Radius
- Shadow Box

Para visualizar los cambios debería ser suficiente importar un tema u otro en el fichero de estilos principal.

## Cómo visualizar los temas
En el fichero ```styles.css``` modificar la ruta del archivo importado en la regla ```@import``` para visualizar un tema u otro:
```
@import url(styles-theme-a.css);
@import url(styles-theme-b.css);
```
