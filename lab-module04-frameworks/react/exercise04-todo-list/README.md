# Cómo ver el proyecto

## Requisitos

- Node.js
- npm

## Cómo ejecutar

1. Clonar el repositorio

```bash
git clone <url-del-repositorio>
cd <nombre-del-proyecto>
```

2. Instalar dependencias

```bash
npm install
```

3. Levantar el servidor en desarrollo

```bash
npm start
```
